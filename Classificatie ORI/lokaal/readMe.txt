"Manual label data" kan gebruikt worden om documenten op te halen vanuit de ORI.API en die met een bootstrap methode te classificeren om een trainingsset te maken.
"Create Database" Kan gebruikt worden om de gelabelde documenten in de SQL database van ORI te zetten.

De tussenversies laten bestanden zien met delen van de classificatie funties die kunnen worden gebruikt ter inspiratie.

Voor de geïmplementeerde bestanden kan je het best kijken op https://github.com/openstate/ori-theme-classifier
